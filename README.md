# vssh
Very simple shell - A SYS-101 Operating Systems Assignment :D 

# Setup
1. Install Rust
2. Clone Git Repository
```
git clone https://github.com/WinsDominoes/vssh
```
3. Enter dir 
4. Run `cargo run --bin vssh`
5. Yay!
